# Demarrage rapide

Bienvenue dans le catalogue Claude AI Tools. Ce guide vous aide a trouver et utiliser les bons outils pour votre workflow.

---

## Qu'est-ce que Claude Code ?

Claude Code est le CLI officiel d'Anthropic pour Claude AI. Il fonctionne dans votre terminal et agit comme un assistant de programmation intelligent capable de lire, ecrire et executer du code dans vos projets. Les outils de ce catalogue etendent ses capacites.

---

## Comment ce catalogue est organise

Les outils sont regroupes en **10 categories** :

| Categorie | Ce que vous y trouverez | Ideal pour |
|-----------|------------------------|------------|
| [Frameworks](../categories/frameworks/) | Methodologies de developpement completes | Structurer l'ensemble de votre workflow de dev |
| [Serveurs MCP](../categories/mcp-servers/) | Connecteurs vers donnees et services externes | Connecter Claude aux BDD, APIs, services cloud |
| [Skills](../categories/skills/) | Jeux d'instructions specialises | Enseigner a Claude des taches ou comportements specifiques |
| [Plugins](../categories/plugins/) | Packages regroupant skills + hooks + commandes | Ameliorations de capacites en un clic |
| [Hooks](../categories/hooks/) | Gestionnaires d'evenements du cycle de vie | Automatisation, securite, controle qualite |
| [Orchestration](../categories/orchestration/) | Outils de coordination multi-agents | Gerer des instances Claude en parallele |
| [Workflows](../categories/workflows/) | Processus de developpement | TDD, CI/CD, pipelines agentiques |
| [Guides](../categories/guides/) | Bonnes pratiques et astuces | Apprentissage et optimisation |
| [Ressources](../categories/resources/) | Repertoires, SDKs, listes awesome | Decouverte et reference |

## Points de depart recommandes

### Si vous debutez avec Claude Code

1. Lisez le guide [Bonnes pratiques CLAUDE.md](../categories/guides/claude-md-best-practices.md)
2. Parcourez les [Astuces et conseils](../categories/guides/tips-and-tricks.md)
3. Installez [Claude Code Templates](../categories/frameworks/claude-code-templates.md) pour des configurations pretes a l'emploi

### Si vous voulez un workflow de developpement complet

1. Commencez par [BMAD Method](../categories/frameworks/bmad-method.md) pour un developpement agile IA complet
2. Ou essayez [GSD](../categories/frameworks/gsd.md) pour un developpement lean, pilote par les specs
3. Ajoutez [Vibe Kanban](../categories/frameworks/vibe-kanban.md) pour la gestion visuelle des taches

### Si vous devez connecter Claude a des services externes

1. Parcourez les [Serveurs MCP](../categories/mcp-servers/) par sous-categorie
2. Commencez par les [serveurs officiels](../categories/mcp-servers/official/) d'Anthropic
3. Consultez les [Registres et repertoires](../categories/resources/registries-directories.md) pour en trouver des milliers d'autres

### Si vous voulez de l'orchestration multi-agents

1. Essayez [Claude Squad](../categories/orchestration/claude-squad.md) pour la gestion en terminal
2. Ou [CCPM](../categories/orchestration/ccpm.md) pour la gestion de projet integree a GitHub
3. Utilisez [Claude Task Master](../categories/orchestration/claude-task-master.md) pour la planification de taches par IA

## Comment lire une fiche outil

Chaque fiche outil suit la meme structure :

1. **Tableau d'en-tete** — Nom, liens, commande d'installation, statut en un coup d'oeil
2. **Ce que ca fait** — Explication claire de l'objectif et de la valeur
3. **Fonctionnalites cles** — Liste a puces des capacites
4. **Installation** — Instructions de mise en place etape par etape
5. **Utilisation avec Claude Code** — Exemples d'integration concrets
6. **Configuration** — Parametres et options
7. **Liens** — Depot, docs, changelog
