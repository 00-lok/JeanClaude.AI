## Ce que ca fait

Le depot `anthropics/skills` est la collection officielle de skills publies par Anthropic pour le standard Agent Skills. Il contient des skills creatifs, techniques et de workflow enterprise qui fonctionnent avec Claude Code et tout agent compatible avec le standard Agent Skills.

Le standard Agent Skills est une specification ouverte adoptee par Microsoft, OpenAI, Cursor, GitHub Copilot, VS Code et 20+ autres plateformes. Un skill cree pour Claude Code fonctionne aussi avec Cursor, Copilot et d'autres agents compatibles.

## Installation

```bash
# Cloner le depot et copier les skills souhaites
git clone https://github.com/anthropics/skills.git
cp -r skills/<nom-du-skill> .claude/skills/

# Ou via SkillKit
npx skillkit install <nom-du-skill>
```
