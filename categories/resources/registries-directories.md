## Hebergement de serveurs MCP

Les serveurs MCP peuvent etre deployes sur :

| Plateforme | Description |
|------------|-------------|
| **Cloudflare Workers** | Edge-first, deploiement en un clic |
| **Vercel** | Integration Next.js, package mcp-handler |
| **AWS Lambda** | Serverless |
| **Smithery.ai** | Deploiement MCP distant heberge |
| **Local / Self-hosted** | La plupart des serveurs supportent le mode STDIO local |
