# Frameworks

Methodologies et systemes de developpement complets concus pour Claude AI. Ce sont les outils les plus complets disponibles — chacun fournit une approche structuree du developpement assiste par IA.

---

| Outil | Description | Stars | Installation |
|-------|-------------|-------|-------------|
| [BMAD Method](bmad-method.md) | Framework agile complet avec 12+ agents IA specialises, 34+ workflows | ~41.8k | `npx bmad-method install` |
| [GSD (Get Shit Done)](gsd.md) | Context engineering et developpement spec-driven avec fenetres de contexte fraiches | ~39k | `npx get-shit-done-cc@latest` |
| [Vibe Kanban](vibe-kanban.md) | Orchestration kanban pour 10+ agents de coding avec apercu integre | ~23.6k | `npx vibe-kanban` |
| [Claude Code Templates](claude-code-templates.md) | 100+ agents, commandes, configs MCP et skills prets a l'emploi | ~23.4k | `npx claude-code-templates@latest` |
| [gstack](gstack.md) | Setup Claude Code de Garry Tan (YC) — 15 outils role-based | ~20k | `git clone` + `./setup` |
| [Spec Kit](spec-kit.md) | Toolkit GitHub pour le developpement spec-driven | — | `bun install -g @spec-kit/cli` |
| [RALPH](ralph.md) | Boucle de dev IA autonome avec detection de sortie intelligente | — | `git clone` + `./install.sh` |

---

## Comment choisir

| Cas d'usage | Recommandation |
|-------------|----------------|
| Simulation d'equipe agile complete (PM, Architecte, Dev, QA, etc.) | **BMAD Method** |
| Projets longs avec gestion de la fenetre de contexte | **GSD** |
| Gestion visuelle des taches avec espaces de travail multi-agents | **Vibe Kanban** |
| Mise en place rapide avec des configurations pre-construites | **Claude Code Templates** |
| Setup opinionated d'un CEO tech | **gstack** |
| Specs comme source de verite (par GitHub) | **Spec Kit** |
| Boucle de dev autonome avec garde-fous | **RALPH** |
| Combiner les frameworks | BMAD + Vibe Kanban ou GSD + CCPM fonctionnent bien ensemble |
