# Orchestration

Outils de coordination multi-agents et de gestion de taches pour Claude Code. Ces outils permettent de gerer plusieurs instances Claude travaillant en parallele, de planifier le travail et de coordonner les resultats.

---

| Outil | Description | Stars |
|-------|-------------|-------|
| [Claude Squad](claude-squad.md) | Application terminal pour gerer des sessions agents en parallele | — |
| [Ruflo](ruflo.md) | Plateforme enterprise de swarm multi-agents avec intelligence auto-apprenante | — |
| [CCPM](ccpm.md) | Gestion de projet avec GitHub Issues + Git worktrees pour execution parallele | ~7.6k |
| [Claude Task Master](claude-task-master.md) | Gestion de taches IA a partir de PRDs, 36 outils MCP | ~25.3k |

---

## Autres outils d'orchestration notables

| Outil | Description |
|-------|-------------|
| [Claude Code Agent Teams (officiel)](https://code.claude.com/docs/en/agent-teams) | Fonctionnalite experimentale native : une session dirige, les autres executent |
| [Gas Town](https://github.com/steveyegge/gastown) | Gestionnaire multi-agents par Steve Yegge, persistance Git |
| [Claude Code Agentrooms](https://claudecode.run/) | App desktop + API pour orchestration via @mentions |
| [Claude MPM](https://github.com/bobmatnyc/claude-mpm) | 47+ agents specialises et 44+ skills en couche d'orchestration |
| Auto-Claude | Framework multi-agents avec UI kanban |
| Claude Swarm | Sessions agents connectees en essaim |
| TSK | CLI Rust delegant des taches a des agents sandboxes |
| [opcode](https://github.com/nichochar/opcode) | GUI desktop Tauri 2 avec editeur visuel d'agents (~21k etoiles) |

## Comment choisir

| Besoin | Recommandation |
|--------|----------------|
| Gestion simple en terminal | **Claude Squad** |
| Enterprise, intelligence distribuee | **Ruflo** |
| Integration GitHub Issues, traceabilite | **CCPM** |
| Planification de taches a partir de PRDs | **Claude Task Master** |
| Fonctionnalite native (experimentale) | **Claude Code Agent Teams** |
