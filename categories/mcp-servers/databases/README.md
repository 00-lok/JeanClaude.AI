# Serveurs MCP — Bases de donnees

Serveurs MCP pour connecter Claude a des bases de donnees relationnelles, NoSQL, vectorielles et analytiques.

---

| Serveur | Description | Type |
|---------|-------------|------|
| PostgreSQL | Acces BDD en lecture seule avec inspection de schema | Relationnel |
| MongoDB | Interaction directe MongoDB, inspection de schema | NoSQL |
| MySQL / MariaDB (via DBHub) | Base de donnees universelle | Relationnel |
| SQLite | Base de donnees embarquee legere | Relationnel |
| Supabase | Base de donnees, auth, edge functions, stockage | Plateforme |
| Neon | Plateforme Postgres serverless | Relationnel |
| Neo4j | Schema et operations lecture/ecriture de graphe | Graphe |
| DuckDB / MotherDuck | Moteur de requetes analytiques | Analytique |
| ClickHouse | Base OLAP orientee colonnes | Analytique |
| Prisma Postgres | Gestion Postgres via ORM | Relationnel |
| SingleStore | Base SQL distribuee | Relationnel |
| Milvus | Base vectorielle pour IA/ML | Vectorielle |
| Chroma | Embeddings, recherche vectorielle, stockage de documents | Vectorielle |
| Redis | Store cle-valeur en memoire | Cle-valeur |
