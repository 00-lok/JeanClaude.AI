# Serveurs MCP — Monitoring et securite

Serveurs MCP pour la surveillance des applications, le suivi des erreurs, l'analyse de securite et la qualite du code.

---

| Serveur | Description | Domaine |
|---------|-------------|---------|
| Sentry | Suivi des erreurs et monitoring | Monitoring |
| Datadog | Plateforme d'observabilite | Monitoring |
| Grafana | Recherche de tableaux de bord et requetes de sources de donnees | Monitoring |
| PagerDuty | Gestion d'incidents | Incidents |
| [Semgrep](https://semgrep.dev/) | Analyse de securite du code | Securite |
| [SonarQube](https://www.sonarsource.com/) | Qualite et securite du code | Qualite |
| Cycode | SAST, SCA, secrets, scan IaC | Securite |
| Raygun | Reporting de crashs et monitoring | Monitoring |
| Logfire | Traces et metriques OpenTelemetry | Observabilite |
