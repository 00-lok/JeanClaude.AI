## Ce que ca fait

Avant d'ajouter une dependance que Claude suggere, Socket la score sur : risque supply chain, qualite, maintenance, vulnerabilites connues, et conformite de licence. Pas de cle API requise pour le serveur public.

---

## Fonctionnalites cles

- **Scoring complet** — Supply chain, qualite, maintenance, vulns, licence
- **Zero config** — Serveur public utilisable sans cle API
- **npm + PyPI** — Ecosystemes JavaScript et Python couverts
- **Prevention proactive** — Avant l'installation, pas apres
