# Serveurs MCP — Cloud et DevOps

Serveurs MCP pour les services cloud, l'infrastructure, les conteneurs et les pipelines CI/CD.

---

| Serveur | Description | Fournisseur |
|---------|-------------|------------|
| [AWS MCP](https://awslabs.github.io/mcp/) | Officiel AWS : Lambda, ECS, S3, EC2, RDS, CDK, Nova, etc. | AWS |
| Azure DevOps | GA depuis nov 2025 : work items, PRs, builds, wikis | Microsoft |
| [Cloudflare](https://developers.cloudflare.com/) | Deployer et configurer des ressources Cloudflare | Cloudflare |
| [Vercel](https://vercel.com/) | Deploiement Next.js avec OAuth, package mcp-handler | Vercel |
| Pulumi | Infrastructure as Code, requetes au registre | Pulumi |
| Terraform MCP | IaC avec Terraform | Communaute |
| Kubectl MCP | Interaction avec cluster Kubernetes (lecture seule par defaut) | Communaute |
| ArgoCD MCP | Gestion de pipelines CD GitOps | Akuity |
| Docker | Gestion de conteneurs | Docker |
| Render | Deploiement de services, requetes BDD, metriques | Render |
| Google Cloud Run | Deployer sur Cloud Run | Google |

---

## Recommandation

Pour le cloud AWS, le serveur **AWS MCP officiel** (~8.5k etoiles) est le choix evident avec sa couverture complete des services AWS.
