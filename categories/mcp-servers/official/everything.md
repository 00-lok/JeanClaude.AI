## Ce que ca fait

Everything est le serveur MCP de reference. Il implemente toutes les fonctionnalites du protocole — prompts, ressources et outils — et sert principalement a tester et valider les clients MCP. C'est le point de reference pour les developpeurs qui construisent des serveurs MCP.

---

## Fonctionnalites cles

- **Implementation complete** — Couvre toutes les capacites du protocole MCP
- **Serveur de test** — Valider le bon fonctionnement des clients MCP
- **Reference** — Modele pour les developpeurs de serveurs MCP
