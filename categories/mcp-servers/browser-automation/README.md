# Serveurs MCP — Automatisation navigateur

Serveurs MCP pour l'automatisation de navigateurs, les tests E2E et l'interaction avec des pages web.

---

| Serveur | Description | Maintenu par |
|---------|-------------|-------------|
| [Playwright MCP](https://github.com/microsoft/playwright-mcp) | Automatisation navigateur officielle Microsoft, snapshots d'accessibilite, 143 appareils | Microsoft |
| Puppeteer | Automatisation navigateur, captures d'ecran, execution JS | Communaute (archive officiel) |
| [Browserbase](https://www.browserbase.com/) | Automatisation navigateur cloud pour extraction de donnees | Browserbase |
| [executeautomation/mcp-playwright](https://github.com/executeautomation/mcp-playwright) | Playwright pour Claude Desktop, Cursor, Cline | Communaute |

---

## Recommandation

**Playwright MCP** de Microsoft est le choix recommande — c'est le plus mature, le mieux maintenu et le plus riche en fonctionnalites.
