# Serveurs MCP — Web scraping et recherche

Serveurs MCP pour le scraping web, l'extraction de donnees et la recherche sur internet.

---

| Serveur | Description | Specialite |
|---------|-------------|-----------|
| [Firecrawl](https://github.com/firecrawl/firecrawl-mcp-server) | Scraping web, crawling par lots, sortie Markdown propre | Scraping |
| Fetch (officiel) | Recuperation et conversion de contenu web | Recuperation |
| [Apify](https://apify.com/) | 3 000+ outils cloud preconstruits pour l'extraction de donnees web | Extraction |
| [Bright Data](https://brightdata.com/) | Donnees web fiables en temps reel ou historiques a grande echelle | Extraction |
| [Tavily](https://tavily.com/) | Moteur de recherche pour agents IA (recherche + extraction) | Recherche |
| [Exa](https://exa.ai/) | Moteur de recherche natif IA | Recherche |
| Kagi Search | Recherche web via l'API Kagi | Recherche |
| [Perplexity](https://docs.perplexity.ai/) | Recherche temps reel via l'API Sonar | Recherche |
| Crawlbase | Recuperation HTML, Markdown, captures d'ecran | Scraping |
